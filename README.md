# lab_sapper
Описание игры:

1. **Общая Концепция**: Игра "Сапер" представляет собой пазл на поле, состоящем из клеток. Цель игры - обнаружить и пометить все мины, спрятанные на поле, не подорвавшись на них.

2. **Игровое Поле**: Поле игры состоит из сетки размером 20x20 клеток. Каждая клетка может быть либо пустой, либо содержать мину. Всего на поле расположено 40 мин, размещенных в случайном порядке.

3. **Взаимодействие с Полем**: Игроки кликают по клеткам, пытаясь определить, где находятся мины. При нажатии на клетку, если она содержит мину, игра заканчивается поражением. Если клетка не содержит мины, она открывается, показывая количество мин в соседних клетках.

4. **Пометка Мин**: Игроки могут помечать клетки, в которых, по их мнению, находятся мины, используя правую кнопку мыши. Это помогает в процессе логического исключения и стратегии.

5. **Логика Игры**: Игра использует функции для определения количества мин вокруг данной клетки и для генерации списка соседних клеток. Это позволяет игроку делать информированные предположения о том, где могут быть мины.

6. **Проверка Победы и Проигрыша**: Игра отслеживает, помечены ли все мины и нет ли ошибочных пометок, для определения победы игрока. При нажатии на мину игра немедленно завершается поражением.

7. **Графический Интерфейс**: Игра использует графический интерфейс, разработанный с помощью Tkinter. Клетки поля отображаются как серые квадраты, которые меняют цвет при взаимодействии.

8. **Взаимодействие с Пользователем**: Игра реагирует на клики левой и правой кнопками мыши. Левая кнопка используется для открытия клеток, а правая - для пометки клеток, предположительно содержащих мину.

Эта версия игры "Сапер" предлагает классический игровой процесс с использованием простого, но эффективного графического интерфейса. В игре используются базовые элементы логики и стратегии, что делает ее интересной как для новичков, так и для опытных игроков.

Также в репозитории раположено видео прохождение игры по вашей просьбе.
